# 项目简介

## 核心功能
- 基于 YOLOv8 的番茄叶病检测模型
- 支持 7 类番茄叶病识别：细菌斑、早期枯萎病、健康、后期枯萎病、叶霉菌、靶点、黑点
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供自定义参数设置：置信度阈值、IOU阈值
- 支持检测结果保存与批量处理
- 实时显示目标位置、目标总数、置信度等信息
- 提供用户友好的图形界面

## 适用场景
- 农业病害检测与研究
- 番茄种植健康监测
- 植物病理学教学演示
- 农业自动化检测系统开发

## 技术特点
- 使用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 基于 YOLOv8 目标检测算法
- 提供完整源码、数据集(737张图片)和训练好的模型
- 包含详细的项目运行教程和界面源文件
- 支持跨平台使用(推荐 PyCharm+Anaconda 或 VSCode+Anaconda 环境)
